# Standalone project sites

pushed to its own GitHub repo and served via GitHub Pages.

| Folder | Suggested URL |
|--------|---------------|
| learning-to-hit      | harshitk95.github.io/learning-to-hit |
| motion-planning-flux | harshitk95.github.io/motion-planning-flux |
| whole-body-hitting   | harshitk95.github.io/whole-body-hitting |
| air-hockey-dual-arm  | harshitk95.github.io/air-hockey-dual-arm |

Every folder is fully portable: it copies its own media and CSS, and links back to the
main site for the home page and publications.
