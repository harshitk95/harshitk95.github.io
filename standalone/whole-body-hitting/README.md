# whole-body-hitting

Standalone GitHub Pages site for the **whole-body-hitting** project, part of
[Harshit Khurana's portfolio](https://harshitk95.github.io/projects.html).

## Deploy as its own site
1. Create a new GitHub repo named `whole-body-hitting`.
2. Copy the contents of this folder to the repo root.
3. In the repo: Settings -> Pages -> Deploy from a branch, select `main` and `/ (root)`.
4. The site goes live at `https://harshitk95.github.io/whole-body-hitting/`.

## Structure
- `index.html` - the page (self-contained)
- `style.css` - portable styles (fonts via Google Fonts CDN)
- `assets/media/` - images and videos
- `assets/paper.pdf` - the paper (where available)
